Introduction¶

This data set contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue. Through this dataset we want to explore :

  1. The relation between budget and revenue.
  2. How budget affects popularity.
  3. How people react to movie runtime.
  4. Old movies or new ones are more popular?
  5. which genres are more popular?

Conclusions¶

  1. Mean revenue is around 108 Million USD
  2. Mean budget is 37 Million USD
  3. Mean vote average is 6.17
  4. Mean runtime is 109 Minutes
  5. Low-budget movies had better vote averages than the higher-budget ones
  6. New movies generated more revenue
  7. 2011 was the highest year in number of movies released while 1969 was the lowest one
  8. Positive correlation between budget and revenue
  9. Positive correlation between budget and popularity
  10. The highest popularity movies have runtime around 100-150 minutes
  11. New movies have higher popularity
  12. The most popular genre among these movies is Drama accounting for 19.9% of all movies while the least popular is Western accounting for about 0.7%

**Analysis limitations:

  1. The genres column wasn't well-organised and each row had more than 1 genre so the analysis was about the times each genre word appear , better if there was a new column with the main genre word
  2. Also there are some meaningless genres as : 'TV movie'
  3. Data was diverse, easy to clean, and sufficient but only for correlations to show up , better if it was more
  4. Column of 'production_companies has sginificant missing values and is badly organised
